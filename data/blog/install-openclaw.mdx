OpenClaw（原名 Clawdbot / Moltbot）是 2026 年初爆火的开源 AI 助手框架，与传统聊天式 AI 不同，它是一个能自主规划、调用工具、执行任务的智能体，可以操作系统、访问网页、处理文件、发送邮件等。

## 前置条件

- Node.js >= 22
- npm 或其他包管理器
- 一个 API Key（支持阿里云百炼、智谱、Kimi、MiniMax 等）

## 安装方式

### 方式一：官方安装脚本（推荐）

**Windows（PowerShell）：**

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

**Windows（CMD）：**

```cmd
curl -fsSL https://openclaw.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

**macOS / Linux：**

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

脚本会自动检测并安装 Node.js（如未安装），完成基础配置。

### 方式二：npm 全局安装

```bash
npm install -g openclaw@latest
```

国内网速慢可以用镜像：

```bash
npm install -g openclaw --registry=https://registry.npmmirror.com
```

### 方式三：阿里云一键部署

如果你有阿里云轻量应用服务器，可以使用应用镜像一键部署：

1. 登录 [阿里云轻量应用服务器控制台](https://help.aliyun.com/zh/simple-application-server/use-cases/quickly-deploy-and-use-openclaw)
2. 选择 OpenClaw 应用镜像
3. 创建服务器即自动完成部署

## 验证安装

```bash
openclaw --version
```

确认版本号 >= 2026.3.8 即可。

## 初始化配置

### 交互式配置（推荐）

```bash
openclaw config wizard
```

配置向导会引导你完成：

- 选择 AI 模型提供商
- 输入 API Key
- 设置默认模型

### 手动配置

编辑配置文件（通常在 `~/.openclaw/config.json`），示例：

```json
{
  "providers": {
    "dashscope": {
      "baseUrl": "https://coding.dashscope.aliyuncs.com/v1",
      "apiKey": "你的百炼API Key",
      "models": ["qwen3.5-coding"]
    }
  },
  "defaultProvider": "dashscope"
}
```

## 启动网关服务

```bash
openclaw gateway start
```

然后就可以开始使用 OpenClaw 了：

```bash
openclaw
```

## 默认模型

- 最新版本默认使用 `qwen3.5-plus` 作为文本和图像处理模型
- 支持切换到其他模型（智谱 GLM、Kimi、MiniMax 等）

## 搭配 Coding Plan 使用

OpenClaw 支持使用各大厂商的 Coding Plan（参阅 [如何订购智谱 GLM、Kimi 和 MiniMax 的 Coding Plan](/blog/coding-plan-glm-kimi-minimax)），以固定月费获取大量 API 调用额度：

- **阿里云百炼 Coding Plan**: OpenClaw 原生深度集成
- **智谱 GLM / Kimi / MiniMax**: 通过配置 API 端点接入
